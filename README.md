# Оформление домашних заданий по курсу Разработчик Bitrix24

## 📌 Описание

Проект **bx24_hw_format** — это репозиторий с примерами оформления домашних заданий по курсу Разработчик Bitrix24.  
Репозиторий содержит необходимые примеры для оформления выполненых заданий.

## 🧠 Что внутри

- `students_dz/` — директория с заданиями, решениями или учебными скриптами студентов.  
- `students_dz/homework2/` — директория домашнего задания №2
- ...

## 🛠 Технологии

**Bitrix24**,  **PHP**

## 🚀 Быстрый старт

### 1. Клонирование репозитория

```bash
git clone https://github.com/SCaeR42/bx24_hw_format.git
cd bx_razr
```
или [Скачать архив с репозиторием](https://github.com/SCaeR42/bx24_hw_format/archive/refs/heads/master.zip)

## 2. Разместить на хостинге

Распоковать скаченый архив в директорию `bx_razr`.

Чтобы получилась следующая структура:

```
/bx_razr/
├── students_dz/
│   ├── homework-1/
│   └── homework-2/
│   └── homework-3/
...
```
